lsst-sdm-tools v30.0.0 (2026-01-16)
===================================

New Features
------------

- Added the ``check-band-colums`` command for checking the self-consistency of band column tables within a schema.
  Added the ``compare-band-columns`` command for comparing band columns between the same tables in two different schemas. (`DM-48803 <https://rubinobs.atlassian.net/browse/DM-48803>`_)
